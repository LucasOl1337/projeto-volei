# Visao do Produto

## Proposito

O Projeto Vôlei sera um aplicativo para auxiliar treinos de volei por meio de registros organizados, relatorios, tabelas e acompanhamento de indicadores de treino.

## Objetivo final

Permitir que um treino gravado seja analisado para identificar aspectos de qualidade, como organizacao da atividade, execucao dos fundamentos, intensidade, volume de repeticoes e evolucao dos atletas.

## Escopo inicial

Nesta primeira fase, o projeto deve focar em:

- Interface inicial do aplicativo.
- Cadastro e anotacao de treinos.
- Registro de relatorios de treino.
- Organizacao de estilos e tipos de treino.
- Tabelas para acompanhar fundamentos, atletas, atividades e observacoes.
- Base de dados simples o suficiente para evoluir depois para analise de video.

## Publico-alvo

- Treinadores de volei.
- Professores de escolinhas ou equipes de base.
- Atletas que acompanham sua propria evolucao.
- Comissoes tecnicas que precisam transformar observacoes de treino em dados consultaveis.

## Principios do produto

- Registro rapido durante ou logo apos o treino.
- Informacao clara para consulta posterior.
- Relatorios objetivos, sem excesso de campos obrigatorios.
- Estrutura preparada para comparacao entre treinos.
- Evolucao gradual para recursos mais avancados, como analise de video.

## Fora do escopo inicial

- Analise automatica de video.
- Reconhecimento de movimentos por inteligencia artificial.
- Sincronizacao com camera ou upload avancado de videos.
- Controle financeiro, agenda completa de competicoes ou gestao administrativa do clube.
