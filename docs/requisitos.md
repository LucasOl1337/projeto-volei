# Requisitos Iniciais

## Modulos principais

### 1. Treinos

O aplicativo deve permitir registrar uma sessao de treino com:

- Data.
- Duracao.
- Local.
- Categoria ou equipe.
- Objetivo principal do treino.
- Estilo de treino.
- Lista de atividades realizadas.
- Observacoes gerais.

### 2. Estilos de treino

O aplicativo deve permitir organizar os treinos por estilo, como:

- Tecnico.
- Tatico.
- Fisico.
- Fundamentos.
- Jogo-treino.
- Recuperacao.

Cada estilo pode ter descricao, foco principal e exemplos de atividades.

Todos os exercicios adicionados ao aplicativo devem ser possiveis de executar individualmente e dentro de casa, usando apenas materiais simples como parede livre, fita, toalha, bola leve, bola de meia, cadeira, cronometro ou espaco curto no chao.

### 3. Relatorios

O aplicativo deve permitir criar relatorios de treino com:

- Resumo do que foi trabalhado.
- Pontos positivos.
- Pontos de atencao.
- Atletas ou grupos em destaque.
- Fundamentos trabalhados.
- Indicadores observados.
- Proximas acoes recomendadas.

### 4. Tabelas e indicadores

O aplicativo deve exibir tabelas para acompanhar:

- Frequencia de treinos.
- Fundamentos treinados.
- Volume de atividades por fundamento.
- Observacoes por atleta.
- Evolucao por periodo.
- Comparacao entre estilos de treino.

### 5. Interface

A interface inicial deve conter:

- Painel principal com resumo dos treinos recentes.
- Tela de cadastro de treino.
- Tela de relatorio de treino.
- Tela de estilos de treino.
- Tela de tabelas ou indicadores.

## Requisitos futuros

### Analise de video

Em uma fase posterior, o aplicativo deve evoluir para:

- Receber videos de treinos.
- Associar um video a um treino registrado.
- Permitir marcacoes manuais em momentos do video.
- Identificar automaticamente eventos ou padroes, quando a base do produto estiver pronta.
- Gerar relatorios a partir de evidencias do video.

## Perguntas abertas

- O aplicativo sera usado por apenas uma pessoa ou por varias contas?
- Os atletas serao cadastrados individualmente desde o inicio?
- O primeiro prototipo deve funcionar localmente ou ja com login e banco online?
- Quais indicadores sao mais importantes para o treino de volei neste momento?
- A interface deve ser pensada primeiro para celular, computador ou tablet?
