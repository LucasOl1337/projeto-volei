Dashboard com indicadores

Use esta regra quando estiver decidindo o que entra no Dashboard do Projeto Isa. O Dashboard responde uma pergunta geral. As telas de detalhe explicam o motivo.

Indicador

Resumo numerico. Deve caber em uma frase e ajudar a decidir onde olhar primeiro.

Exemplo: Fisico 7.2, Tecnico 8.1, Fundamento 6.9.

Tela de detalhe

Investigacao. Mostra tabelas, listas, historico, exercicios e observacoes.

Exemplo: se Fundamento esta baixo, abrir biblioteca, indicadores ou relatorios.

Regra pratica

Se precisa rolar muito, nao e Dashboard. Leve para uma tela especifica.

O Dashboard deve ficar escaneavel em poucos segundos.
