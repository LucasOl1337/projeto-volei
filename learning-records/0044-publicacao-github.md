# 0044 - Publicacao do estado atual no GitHub

## Dominio do volei

Este commit funciona como um relatorio fechado de treino: ele guarda o estado atual do app, dos criterios, dos exemplos de evidencia e dos materiais de aprendizagem. Assim, cada evolucao futura pode ser comparada com uma versao anterior.

## Conceito de desenvolvimento

Um repositorio GitHub guarda o historico do projeto. O commit e o registro de uma versao especifica; o `.gitignore` define o que nao deve entrar nesse historico, como dependencias instaladas e arquivos temporarios.

## Decisao

O Projeto Volei sera publicado como repositorio publico para preservar o MVP navegavel, a documentacao de produto e os estudos de evidencia manual antes de avancar para recursos mais complexos de video e IA.
