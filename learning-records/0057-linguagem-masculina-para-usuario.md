# 0057 - Linguagem masculina para usuario

No Projeto Isa, a interface e os materiais de apoio devem tratar o usuario com pronomes masculinos.

Decisao: usar `o usuario`, `o atleta`, `um atleta`, `do atleta` e formas equivalentes. A palavra `atleta` pode continuar sem artigo quando funcionar como termo tecnico ou categoria.

Conceito de desenvolvimento: texto tambem e parte da experiencia do produto. Quando a linguagem nao corresponde ao usuario, a interface parece menos pessoal, mesmo que a funcao esteja correta.
