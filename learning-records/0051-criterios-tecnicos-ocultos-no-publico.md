# Criterios tecnicos ocultos no publico

Data: 2026-07-01

No treino de volei, o atleta pode precisar ver resultado, evidencia e proxima acao sem receber todos os criterios internos de revisao. Os criterios continuam importantes para o treinador e para o review, mas nao precisam aparecer como uma lista publica na tela principal.

Conceito de desenvolvimento: separar dado de interface. O dado pode continuar existindo para calculo, calibracao e relatorio; a interface decide se esse dado deve ou nao ser mostrado para a pessoo usuario naquele momento.

Decisao: a tela de analise de video deixou de renderizar o painel publico `Revisao tecnica`. As evidencias, metricas e criterios seguem no fluxo interno de processamento, calibracao e uso no relatorio.
