# Projeto Isa

Aplicativo de apoio a treinos de volei, com foco inicial em registrar sessoes, organizar relatorios e montar tabelas que ajudem a acompanhar a qualidade e a evolucao dos treinos.

## Objetivo

Construir uma ferramenta simples e pratica para treinadores e atletas registrarem treinos de volei, classificarem estilos de treino, acompanharem indicadores e, futuramente, analisarem videos gravados para avaliar a qualidade das atividades.
